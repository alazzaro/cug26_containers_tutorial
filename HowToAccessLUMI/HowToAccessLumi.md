
# Get Started

(based on https://docs.lumi-supercomputer.eu/firststeps/)


1. Get a user account (https://docs.lumi-supercomputer.eu/firststeps/accessLUMI/)

2. Set up an SSH key pair (Needed, if you are using LUMI from a terminal)

3. Log in to LUMI with SSH client or with the web interface

If applicable, move your data to LUMI

Explore your options for using LUMI
